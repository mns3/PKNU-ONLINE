## 🎮 프로젝트 소개
**PKNU-ONLINE**은 생성형 AI와 절차적 생성(PCG) 알고리즘을 활용하여 **디지털 트윈 콘텐츠 생성의 자동화 파이프라인**을 구축하고, 이를 통해 부경대학교 캠퍼스를 가상 공간에 완벽하게 재현한 프로젝트입니다.

브이월드(V-World) GIS 데이터를 기반으로 실제 지형을 구성하고, 현장 사진을 생성형 AI에 학습시켜 고품질 텍스처와 모델을 자동으로 생성했습니다. 단순한 모델링 작업을 넘어, 기술을 통해 대규모 환경 제작의 효율성을 극대화하였으며 멀티플레이 환경을 통해 커뮤니티 기능을 제공합니다.

## 🛠 기술 구현 상세 (Technical Implementation)

### 1. 생성형 AI 기반 자동화 파이프라인 (Automated Pipeline with Gen-AI)
* **GIS 데이터와 AI의 결합:**
    * 브이월드(V-World)의 GIS 데이터를 기반으로 LoD1(Level of Detail 1) 수준의 기본 매스 모델을 생성하여 정확한 스케일을 확보함
    * 현장에서 촬영한 사진 데이터를 생성형 AI에 학습 및 적용하여, 단순한 지형 데이터를 고품질의 텍스처와 디테일한 3D 모델로 변환하는 자동화 워크플로우를 구축함

### 2. 절차적 콘텐츠 생성 및 최적화 (PCG & Optimization)
* **대규모 환경의 절차적 생성 (PCG Framework):**
    * 수작업 배치의 비효율을 해결하기 위해 언리얼 엔진의 PCG 프레임워크를 도입
    * 가로수, 벤치, 구조물 등 반복적인 오브젝트를 알고리즘 기반으로 자동 배치하여 대규모 캠퍼스 환경을 효율적으로 제작함
* **나나이트(Nanite) 가상화 지오메트리:**
    * 고해상도 에셋을 성능 저하 없이 렌더링하기 위해 나나이트 기술을 적용, 수백만 폴리곤의 디테일을 실시간으로 구현함

### 3. 디지털 트윈 및 멀티플레이 환경 (Digital Twin & Networking)
* **실감형 캠퍼스 구현:**
    * 실제 부경대학교의 지형 정보와 건축물 데이터를 1:1 스케일로 매핑하여 현실감 있는 디지털 트윈을 완성함
* **멀티플레이어 커뮤니티:**
    * 다수의 유저가 동시에 접속하여 상호작용할 수 있는 네트워크 환경을 구축하여 메타버스 커뮤니티 기능을 제공함

## 💡 프로젝트 회고 및 배운 점 (Retrospective)

* **자동화 파이프라인 구축:** 생성형 AI를 활용한 에셋 제작 자동화 파이프라인을 직접 설계하고 구축하며, 기존 수작업 방식 대비 비약적인 생산성 향상을 경험함
* **데이터 파이프라인 관리:** GIS 데이터와 현장 사진 등 이기종 데이터를 확보하고 가공하여 엔진에 통합하는 과정에서 대용량 리소스 관리 및 데이터 파이프라인 처리 능력을 배움
* **최신 엔진 기술 활용:** 언리얼 엔진 5의 핵심 기술인 **PCG(절차적 생성), 나나이트(Nanite), 루멘(Lumen)**을 프로젝트에 주도적으로 적용하며 차세대 엔진 활용 능력을 확보함
* **TA 역량 강화:** 단순한 맵 제작이 아닌, '현실 데이터의 디지털화'라는 관점에서 기술적 문제를 해결하며 테크니컬 아티스트(TA)로서의 시야를 넓힘
